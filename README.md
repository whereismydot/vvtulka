<h1 align="center">🧾 VVTulka</h1>
<p align="center"><b>Локальный офлайн-инструмент</b> для разбора чеков, расчёта ВкусБэк и быстрых утилит для текста и срока годности.</p>

---

## ✨ About

VVTulka — это набор небольших прикладных инструментов, которые работают прямо в браузере и сохраняют данные локально.

### 🔎 Что уже есть
- Калькулятор ВкусБэк по табличному чеку.
- Очистка текста в реальном времени.
- Калькулятор срока годности.

## 🚀 Локальный запуск

### 1) Установка зависимостей
```bash
npm install
```

### 2) Dev-режим
```bash
npm run dev
```
