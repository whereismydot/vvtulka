# VV Local Tool

Статический офлайн-инструмент для разбора чеков и расчета ВкусБэк/кэшбэка прямо в браузере.

## Возможности

- Вставка чека из TAB-таблицы (как есть из источника).
- Фильтрация товаров по полю `ВходитВкусБэк`.
- Точные денежные расчеты через `decimal.js` (без `Number` для денег).
- Локальное хранение заказов и процента в `localStorage`.
- Копирование итогового кэшбэка в буфер обмена.
- Редактирование/копирование названия заказа.
- Переключение светлой/темной темы.
- Кнопка быстрого возврата наверх при прокрутке.

## Технологии

- Vite
- TypeScript
- decimal.js
- Vitest

## Локальный запуск

```bash
npm install
npm run dev
```

## Проверка и тесты

```bash
npm run test
npm run build
```

`npm run build` включает `tsc --noEmit`, поэтому одновременно проверяет типы и сборку.
