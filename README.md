# VV Local Tool

Локальный офлайн-инструмент для разбора табличных чеков и расчёта ВкусБэк/кэшбэка в браузере.

## Назначение проекта

Приложение помогает быстро:
- выделить позиции, которые участвуют в ВкусБэк;
- посчитать сумму ВкусБэк и итоговый кэшбэк по проценту;
- хранить историю заказов локально в `localStorage`.

## Установка зависимостей

```bash
npm install
```

```bash
npm run dev
```
